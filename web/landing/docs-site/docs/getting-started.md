# Getting started

## Install

Use the download section on the landing page to pick your platform package.

## Build from source

```bash
git clone https://github.com/usetonet/tonet-browser.git
cd tonet-browser
cargo build --release -p tonet
```

## Runtime basics

- Open `tonet://settings` to configure update policy, language, and system behavior.
- Open `tonet://downloads` to inspect downloaded and saved content.
- Open `tonet://history` to review visits and clear browsing data.
