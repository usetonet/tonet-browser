# Tonet documentation

Tonet documentation is now organized as a structured portal with clear sections for users, operators, and contributors.

## What you will find

- Product overview and design principles.
- Installation and packaging references.
- CDN-based release delivery and update manifest setup.
- Use cases, comparisons, and implementation roadmap.

## Project positioning

Tonet prioritizes transparency and deterministic behavior over broad "everything works" compatibility claims.  
The project publishes roadmap and quality gates so changes remain measurable and auditable.
