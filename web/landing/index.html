Tonet es un navegador en construcción con una filosofía clara: velocidad extrema, ligereza absoluta y un motor propio para contenido esencial. Sin Blink, WebKit ni CEF — tú controlas qué entra.

Descargar Tonet Ver documentación

Diseñado para ir al grano

Un MVP honesto: red + parser + render mínimos. Tonet no pretende renderizar la web industrial de hoy; pretende devolver la lectura y la búsqueda a algo humano.

Filtro de pureza

Páginas mayores de 1 MB se rechazan. Tonet solo carga lo que cabe en un estándar de ligereza.

Parser propio

Extracción manual de título y texto esencial — sin arrastrar un motor HTML completo.

Actualizaciones bajo tu control

Comprueba novedades al iniciar, cada 24 h o solo cuando tú quieras. Un clic abre las descargas oficiales.

Descargar Tonet

Detectamos tu sistema para resaltar la opción adecuada. Siempre puedes cambiar de pestaña manualmente.

Windows

Instaladores generados en CI: paquete MSI (empresas / InTune) y EXE con Inno Setup.

Descargar MSI (x64)   Descargar EXE (x64)

Tras publicar un release, los archivos aparecen como Tonet-<versión>-x64.msi y Tonet-Setup-<versión>-x64.exe. La página de «latest» lista siempre los binarios vigentes.

Linux

Paquete .deb para amd64 (cuando el workflow de release lo publique) o compilación desde fuente.

Abrir releases (.deb)

Compilar desde el repositorio

git clone https://github.com/usetonet/tonet-browser.git
cd tonet-browser
cargo build --release -p tonet
./target/release/tonet

Instalación de usuario (desktop + binario)

cargo build --release -p tonet
chmod +x scripts/install-linux-user.sh
./scripts/install-linux-user.sh

macOS

El bundle .app firmado y notarizado está en la hoja de ruta. Mientras tanto puedes compilar con Rust estable y Xcode Command Line Tools. 

git clone https://github.com/usetonet/tonet-browser.git
cd tonet-browser
cargo build --release -p tonet
open target/release/tonet
