Browse without the weight.
Push back on web bloat.

Tonet is a browser built with a clear goal: extreme speed, minimal weight, and an in-house engine for essential content. No Blink, WebKit, or CEF — you control what comes in.

Download Tonet Read the docs

Built to get to the point

An honest MVP: network + parser + minimal renderer. Tonet does not try to render today’s industrial web; it tries to make reading and search feel human again.

Purity filter

Pages over 1 MB are rejected. Tonet only loads what fits a lightweight standard.

Custom parser

Manual extraction of title and essential text — without dragging in a full HTML engine.

Updates on your terms

Check on launch, every 24 hours, or only when you want. One click opens official downloads.

Open source

Full transparency on GitHub. Signed installers and documented pipelines for operators.

Debian package (.deb)

Build from the repository

git clone https://github.com/usetonet/tonet-browser.git
cd tonet-browser
cargo build --release -p tonet
./target/release/tonet

Per-user install (desktop + binary)

cargo build --release -p tonet
chmod +x scripts/install-linux-user.sh
./scripts/install-linux-user.sh
