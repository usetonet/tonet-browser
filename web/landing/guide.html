Using Tonet

This page is for everyday users—no packaging jargon, no CI pipelines. Developers looking for signing scripts and manifests should open Technical documentation instead.

Install Tonet

Go to Downloads. Your OS tab is highlighted automatically: use the orange button to fetch the installer from our CDN while you stay on the page. When the file finishes, run it like any other desktop app.

Stay up to date

Tonet can check for updates from the official manifest. Open Settings (gear icon) and choose how often you want it to check—on launch, every day, or only when you tap “Check now”. Tonet tells you when a build is ready; you choose when to download it.
