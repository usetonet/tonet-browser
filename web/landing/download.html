Linux

Primary download is the CI-built .deb for amd64 (Debian/Ubuntu) matching the version embedded on this site. Other layouts and source builds are below.

Download Tonet (Linux)

Debian package (.deb)

Build from the repository

git clone https://github.com/usetonet/tonet-browser.git
cd tonet-browser
cargo build --release -p tonet
./target/release/tonet

Per-user install (desktop + binary)

cargo build --release -p tonet
chmod +x scripts/install-linux-user.sh
./scripts/install-linux-user.sh

macOS

macOS binaries (tarball or TonetSetup) appear on GitHub Releases when published for a tag. Until then, build from source below. A signed .app bundle is on the roadmap.
